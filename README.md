Grocery Management System

📌 Overview

A desktop-based Grocery Management System developed using Core Java, Swing, and MySQL.
The application allows users to register and log in, select grocery items, generate a bill, store order details in a database, and download the bill receipt.

This project simulates a real-world retail billing system and demonstrates GUI development with database integration.

---

🚀 Features

- User Registration and Login Authentication
- Grocery Product Selection Interface
- Order Summary Page
- Automatic Bill Calculation
- Order Data Storage in MySQL Database
- Downloadable Bill Receipt (.txt)

---

🛠 Technologies Used

- Java (Core Java)
- Java Swing (GUI Development)
- JDBC (Database Connectivity)
- MySQL

---

💻 How to Run the Project

1. Clone the repository
2. Import the project into Eclipse or IntelliJ IDEA
3. Create the database using the "database.sql" file
4. Update database credentials in "DBConnection.java"
5. Run "MainLogin.java"

---

📌 Project Description

The Grocery Management System allows users to select grocery products, generate bills, and store order details in a MySQL database. The system also provides an option to download the generated bill receipt.

The project demonstrates:

- Java Swing GUI design
- Database connectivity using JDBC
- CRUD operations with MySQL
- File handling for receipt generation
