# Grocery Management System

## 📌 Overview
A desktop-based Grocery Management System developed using Core Java, Swing, and MySQL.

## 🚀 Features
- User Login Authentication
- Product Selection
- Order Summary
- Real-time Bill Calculation
- MySQL Database Integration using JDBC
- Downloadable Bill Receipt

## 🛠 Technologies Used
- Java
- Swing
- JDBC
- MySQL

## 💻 How to Run
1. Clone the repository
2. Import into Eclipse/IntelliJ
3. Configure MySQL database
4. Run Main.java
